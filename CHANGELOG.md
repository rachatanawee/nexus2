# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-XX

### Added
- Next.js 16 with App Router
- Supabase authentication with cookie-based JWT
- Role-Based Access Control (RBAC)
- Multi-language support (EN/TH) with next-intl
- Admin dashboard with collapsible sidebar
- User management with full CRUD operations
- Inventory management (Products & Warehouses)
- tnks-data-table integration with search, sort, pagination, export
- API routes for external system integration
- Feature-Colocation architecture pattern
- Tailwind CSS v4
- shadcn/ui components

### Security
- API key authentication for external APIs
- Row Level Security (RLS) ready
- JWT-based session management
